如何在VS2010中运行C语言程序

字数统计: 621 字   |   阅读时长: 2 分   |   本文总阅读量: 次


C/C++
  • 计算机二级C语言

本文主要给小白讲解如何在VS2010环境下运行C语言程序，最近很多大一新生开始准备全国计算机二级了，对于C语言来说的话，不再使用老牌的VC6.0，自2018年开始就使用VS2010环境了（学习版），故写一篇文章讲解一下，其实我是想锻炼一下好久没写的Markdown（逃

  1. 第一步，下载VS2010
    实际上你可以到学校机房办公室找老师要一个安装包，当然，百度一下资源一大堆，我这里提供一个：VS2010Express 密码：bloh

  2. 第二步，打开vs2010，点击新建项目

  3. 第三步，点击选 WiN32控制台应用程序，别忘了要在下方输入名称，这个随便起，我这里给的是demo

  4. 然后会出现两个窗口，按照图示依次操作即可，这里有主意你必须选择的是控制台应用程序的空项目

  1. 接着右击选中源文件——>添加——>新建项

  2. 然后选中C++文件——>记得把名称改为以.c为后缀的文件（因为这里你要运行的是C文件)——>点击添加

  3. 下面就像VC6.0一样在里面输入C语言代码就可以了，如图，按图示顺序编辑好后一定记得保存，再点击调试或者图形快捷窗口，当然还有快捷键F5。按过后会出现一个对话框，我们直接点击是，最好勾上不再提示，就可以运行结果了。

  4. 点击后就会出现一个结果啦，当然，由于我们的代码中没有添加system("pause");暂停语句，点击“运行”按钮，或者按下F5键后程序只会一闪而过，只能看到一个“黑影”。如果想让程序自动暂停，可以按下Ctrl+F5组合键，这样程序就不会一闪而过了，换句话说，按下
    Ctrl+F5键，VS2010会自动在程序的最后添加暂停语句。

至此，咱们就可以在VS2010里运行C语言程序啦，实际上C++是C语言的超集，在第6步里不改后缀名也不会影响一些程序的运行。C语言还有更多值得探索的地方，后面我们会提到，我希望能做一个二级C语言的系列教程，这只是个开端，希望这篇教程可以对你有所帮助。
